Notes:

- Some verbs appear in the text only in one aspect (especially perfectives in dialogue)

- "Есть" (there is/are) is a special present tense form of "быть"

- A few verbs like "высыпать" have identical imperfective and perfective forms

- Iterative verbs like "бегать" (to run around) differ from directional "бежать" (to run in a direction)

- Some entries may be repeated where the verb appears multiple times in different contexts
